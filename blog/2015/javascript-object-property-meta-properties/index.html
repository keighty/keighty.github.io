Javascript Object Property Meta Properties

JavaScript Objects are complex types, and even defined properties have a control panel of their own. You can view the content of the control panel using Object.getOwnPropertyDescriptor(), and flip their switches using Object.defineProperty().

Note: These property descriptors are for own properties — that is, properties that are directly defined on an object, not those that come along for the ride on the object’s prototype chain.
